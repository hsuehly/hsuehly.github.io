数据库知识三

Hsueh 2021-03-06 数据库mysql

# 1.列约束

（1）唯一约束——unique

声明了唯一约束的列上不允许重复，允许插入null,允许插入多个null，null代表一个无法确定的值，两个null、不能认为是相同的

（2）默认值的约束——default

可以使用default关键字来设置默认值，具体应用默认值有两种

insert into family values(40,default);
insert into family(fid) values(50);#给特定的列值，没有出现的列会使用默认的值
1
2

（3）检查约束——check

自定义约束。可以自己指定约束的条件

create table student(
score tinyint check(score>=o and score<=100)
);
1
2
3

mysql 不支持检查约束，认为严重影响数据的插入速度后期由js代替

（4）外键约束——foreign key

声明了外键约束的列，取值范围到另一个表的主键列；如果不存在则禁止插入。写在最后面

  • 允许插入null
  • 外键列和对应表的主键列，列类型要保持一致。
foreign key (外键列) references 另一个表(主键列)
1

# 2.自增列

auto_increment: 自动增长，如果设置了自增列，只需要赋值为null,就会在原来的基础上加1插入

自增列必须加在主键形式的编号列上

允许手动赋值y

# 3.简单查询

（1）查询特定的列

示列：查询所有员工的编号和姓名

select eid,ename from emp;
1

练习：查询所有员工的姓名，性别，生日，工资

select ename,sex,birthday,salary from emp;
1

(2)查询所有的列

select eid,ename,sex,birthday,salary,deptId from emp;
select * from emp;
1
2

(3)给列起别名

示列：查询出所有员工的编号和姓名，要求使用汉字别名

select eid as 编号,ename as 姓名 from emp;
1

练习：查询所有员工的姓名，生日，工资

select ename a,birthday b,salary c from emp;
1

起别名是为了简化列名称；as 关键字可以省略，中间保留空格即可。

（4）显示不同的记录

示列：查询出都有那些性别的员工

select distinct sex from emp;
1

练习：查询出员工都很不在那些部门

select distinct deptId from emp;
1

(5) 查询时执行计算

示列： 计算5+3*2.7*3.9+85.77

select 5+3*2.7*3.9+85.77;
1

练习：查询出所有员工的姓名及其年薪

select ename,salary*12 from emp;
1

练习：假设每个员工的工资加2000,年终奖30000,查询出所有员工的姓名及其年薪，使用汉字作为别名

select ename 姓名,(salary+2000)*12+30000 年薪 from emp;
1

（6）查询的结果集排序

示列：查询出所有的部门，结果集按照部门编号升序排列

select *from dept order by did asc;
1

示列：查询出所有的部门，结果集按照部门编号降序排序

select * from dept order by did desc;#descendant 降序
1

练系：查询出所有的员工，结果集按照工资降序排列

select * from emp order by salary desc;
1

练习：查询所有的员工结果集按照年龄从大到小排列（生日从小到大）

select * from emp order by birthday asc;
1
如果不加排序规则默认按照升序排列。
如果按照字符串排序，按照首个字符的Unicode码来排列
1
2

练习：查询出所有员工，结果集按照姓名升序排列

select * from emp order by ename;
1

练习：查询出所有的员工，结果集按照工资的降序排列，如果工资相同按照姓名排列；

select * from emp order by salary desc,ename;
1

练习：查询所有员工，要求女员工显示在前边，如果性别相同按照年龄从小到大排列

select * from emp order by sex asc,birthday desc;
1

(7)条件查询

示列：查询编号为5的员工

select *from emp where eid=5;
1

练习：查询出姓名为lucy的员工

select * from emp where ename='lucy';
1

练习：查询出所有的女员工

select * from emp where sex=0;
1

练习：查询工资在5000以上的员工有那些

select * from emp where salary>5000;
1

比较运算符 > < >= <= = !=(不等于)

练习： 查询出不在一号部门的员工

select * from emp where deptId!=1;
1

练习：查询出没有明确部门有那些

select * from emp where deptId is null;
1

找null不能用= 要用is

练习：查询出有明确部门员工有那些

select * from emp where deptId is not null;
1

练习：查询出工资在5000-8000之间的员工有那些

select * from emp where salary>=5000 and salary<=8000;
select * from emp where salary between 5000 and 8000;
1
2

练习： 查询出1993年出生的员工有那些

select * from emp where birthday>='1993-1-1' and birthday<='1993-12-31';
select * from emp where birthday between '1993-1-1' and '1993-12-31';
1
2

练习：查询出工资在5000 以下或者8000以上的员工有那些

select * from emp where salary<5000 or salary>8000;
select * from emp where salary not between 5000 and 8000;
1
2

练习：查询出2号部门或者3号部门的员工有那些

select * from emp where deptId=2 or deptId=3;
select * from emp where deptId in(2,3);
1
2

练习：查询出不在2号或者不在3号部门的员工有那些

select * from emp where deptId not in(2,3);
1

(8)模糊条件查询

示列： 查询出姓名中含有字母e的员工有那些

select * from emp where ename like '%e%';
1

练习：查询出姓名中以e结尾的员工有那些

select * from emp where ename like '%e';
1

练习：查询出姓名中倒数第二个字符是e的员工

select * from emp where ename like '%e_';
1

% 匹配任意0 个或者多个字符 >=0

_ 匹配任意一个字符

以上两个平儿配符号必须结合着like关键词使用

# 4.分页查询（简单查询）

查询的结果集中有太多的数据，一次显示不完可以做成分页显示

需要提供两个条件：当前的页码，每页的数据量

每页开始查询的值 = （当前页码-1） 每页的数据量*

select * from emp limit 开始查询的值,每页的数据量；
1

练习： 每页显示5条数据，分别查询出前3页的数据

第一页：

select * from emp limit 0,5;
1

第二页：

select * from emp limit 5,5;
1

第二页：

select * from emp limit 10,5;
1

limit 后的两个值是数值型，不能加引号

练习 ：查询出所有男员工工资最高的前3个人的姓名，生日，工资。

select ename,birthday,salary from emp where sex=1 order by salary desc limit 0,3;
1

(2)复杂查询

（1）聚合查询/分组查询

示列：查询出所有员工的数量

select count(*) from emp;
1

练习： 使用员工的编号查询所有男员工的数量

select count(eid) from emp where sex=1;
1

练习：使用部门的编号查询所有男员工的数量

select count(deptId) from emp where sex=1;
1

推荐使用主键列来计算数量

聚合函数：

函数：是一个独立的功能体，需要提供若干个数据，返回结果

count() / sum() / avg() / max() / min()

数量 总和 平均 最大 最小

练习：计算出所有员工的工资总和

select sum(salary) from emp;
1

练习：计算所有女员工的平均工资

select avg(salary) from emp where sex=0;
1

练习：计算出1号部门员工的最低工资

select min(salary) from emp where deptId=1;
1

练习：计算出年龄最小的员工生日是多少

select max(birthday) from emp;
1
分组查询通常只查聚合函数和分组条件
1

示列：分别查询出男女员工的数量，最高工资，最低工资

select count(eid),max(salary),min(salary),sex from emp group by sex;
1

练习：查询出各个部门员工的工资总和，平均工资。

select sum(salary),avg(salary),deptId from emp group by deptId;
1

示列：查询‘2021-3-4’

select year('2021-3-4');
1

练习：查询出1993年出生的员工有那些

select * from emp where year(birthday)='1993';
1

year() 获取日期中的年份

month()获取日期中的月份

（2）子查询

示列：查询出工资最高的员工所有的列

步骤1：查询出最高工资

select max(salary) from emp;
1

步骤2：通过最高工资的值查询出员工

select * from emp where salary=22000;
1

综合：

select * from emp where salary=(select max(salary) from emp);
1

练习：查询出比tom工资高的员工有那些

select * from emp where salary>(select salary from emp where ename='tom');
1

练习：查询出和tom同一年出生的员工有那些

select * from emp where year(birthday)=(select year(birthday) from emp where ename='tom') and ename!='tom';
1

子查询是多个SQL命令的组合，把一个命令的结果作为另一个SQL命令的条件。

（3）多表查询

查询的数据分布在多个表中

示列：查询出所有的员工姓名及其部门名称

select emp.ename ,dept.dname from emp,dept where emp.deptId=dept.did;
1

多表查询的前提是在创建表的时候已经建立了关联，一个表外键对应另一个表的主键

内连接

select ename,dname from emp inner join dept on deptId=did;
                            #内连接           #条件
1
2

左外连接

select ename,dname from emp left outer join dept on deptId=did;
1

左侧表中所有的记录都显示即使没有对应的数据，先写那个表那个就是左，其中outer关键字可以省略

右外连接

select ename,dname from emp right join dept on deptId=did;
1

右侧表中所有的数据都显示即使没有对应的数据，后写那个表那个就是有 outer关键字可以省略
